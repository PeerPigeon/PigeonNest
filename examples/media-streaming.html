üìñ How to Use

  1. Open this page in two different browser windows (can be same or different browsers)
  2. Enter different Peer IDs for each window (e.g., "peer1" and "peer2")
  3. Use the same Network Name in both windows (default: "media-demo")
  4. Click Connect in both windows
  5. Wait until both peers show "Connected: 1" in the Connected Peers field
  6. In one window, select a media file and enter the other peer's ID
  7. Click Send Media and watch it stream to the other peer!
